# 欢迎使用 夏午茶 后台管理系统

------

介绍：夏午茶后台管理系统是一个基于YII2框架的后台,采用 boosstrap 布局渲染
开发背景：强迫症患者，不喜欢YII封装的CURD等，喜欢自己自定义，精简项目冗余代码,功能中穿插YII各种特性行为 事件,widgets，php traint等


第一版目前提供了如下功能：
- [x] 用户登陆
- [x] 图片上传
- [x] 富文本
- [x] 文章CURD
- [x] 权限控制
- [x] 操作日志
- [ ] 其他 (开发中)
